# mineral.web

mineral.web とは、WebComponent で作成されたコンポーネント集です。
あなたの Web にタグを記述するだけで、実装を行うことができます。

- 静的開発、動的開発ともに利用することができます。
- 純粋なのでカスタムを行いやすいです。
- Shadow DOM を使用しているので、スタイルが衝突しません。
