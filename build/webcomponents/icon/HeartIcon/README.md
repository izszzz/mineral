# HeartIcon

## Demo

```html
<heart-icon></heart-icon>
```

## Attribute

| summary           | attribute    | value | default | type   |
| ----------------- | ------------ | ----- | ------- | ------ |
| icon size         | size         |       | 100     | number |
| icon fill color   | fill         |       | none    | color  |
| icon stroke color | stroke       |       | red     | color  |
| icon stroke width | stroke-width |       | 5       | color  |
