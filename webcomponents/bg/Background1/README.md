## Attribute

| summary       | attribute | value | default     | type   |
| ------------- | --------- | ----- | ----------- | ------ |
| canvas width  | width     |       | window size | number |
| canvas height | height    |       | window size | number |
| make polygon  | number    |       | 100         | number |
| polygon color | colors    |       | empty       | string |
